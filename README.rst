Plone OSX Installer
===============================================

Overview
--------

    This is the development environment needed to create the OSX Installer for Plone.

Requirements
------------

    * MacOSX 10.7.x
    
    * XCode 4.3 (https://developer.apple.com/)

    * Auxiliary Tools for XCode (https://developer.apple.com/)
